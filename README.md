# upimg-cli.js

需要自行安装nodejs和git，旨在快速上传图片文件到远程仓库，便于在任意地方使用。

## 全局安装

```
    npm i -g upimg-cli
```


## 初始化

```
    upimg init
```

***注意：若使用gitee（码云）创建仓库时需要将仓库设为public，否则无法查看图片。***


## 上传图片

```
    upimg [path]
```

上传成功后直接粘贴链接到需要使用的地方即可。

## 重置本地仓库
```
    upimg reset
```
远程仓库已满或者需要更换仓库时，可以执行该命令，重新配置。
